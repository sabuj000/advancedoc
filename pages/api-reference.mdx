# 📘 Create a Campaign API

Use this endpoint to create a new Campaign.

## Sample Request

<ApiRequest>
{`
{
  "campaign": {
    "name": "Test",
    "goal_ids": [101],
    "status": "running"
  }
}
`}
</ApiRequest>
