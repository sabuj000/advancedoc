# Command: Create KB Article

## What it does

Produces a troubleshooting article using the `kb-article` skill: define the problem and root cause, write imperative steps, and output a file under `kb/articles/` that follows `templates/kb-article.md`.

## When to use

- A recurring support issue needs a documented fix.
- You are publishing a workaround for a known defect.
- You need consistent structure for search and AI retrieval.

## How to run (manual v1)

1. Read `docs-system/.docs/skills/kb-article/SKILL.md`.
2. Draft using `templates/kb-article.md`.
3. Validate with `checklist.md` and `style/user-style.md`.
