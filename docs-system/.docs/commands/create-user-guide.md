# Command: Create User Guide

## What it does

Creates a procedural user guide with overview, prerequisites, numbered steps, and result using the `user-guide` skill and `templates/user-guide.md`, saved under `user-docs/guides/`.

## When to use

- Shipping or updating a feature customers complete in the UI.
- Adding onboarding content that must match product labels.
- Linking from marketing or in-app help to detailed steps.

## How to run (manual v1)

1. Read `docs-system/.docs/skills/user-guide/SKILL.md`.
2. Copy `templates/user-guide.md` and fill all sections.
3. Validate with `checklist.md` and `style/user-style.md`.
